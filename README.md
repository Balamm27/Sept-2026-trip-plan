# September 2026 Trip Planner

Shareable static dashboard for planning a September 2026 hill-station trip from Chennai.

## What is inside

- A mobile-friendly single-page dashboard
- Trip option ranking for Coonoor, Yercaud, Yelagiri, and Kodaikanal
- A small interactive budget and trip-style selector
- A recommended 3-night itinerary

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Save. GitHub will publish the site and give you a shareable URL.

If Pages is already enabled, pushing these files is enough.
